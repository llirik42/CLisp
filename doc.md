# CLisp documentation

The interpreter made according to the R5RS scheme.

## IO

### display

```Scheme
(display "Hello, world!")
```

```
Hello, world!
```

> unlike the R5RS, `display` adds ``\n`. 

> `display` accepts only one argument.

### newline

```Scheme
(newline)
```

> `(newline)` is equivalent to `(display "\n")`.
